My Goals

These programs are designed to be a fun introduction to the basics of programming in Python. The focus will be on building simple games such as Pong, Blackjack and Asteroids. All files posted here were create for the the Coursera course 'An Introduction to Interactive Programming in Python'. The focus will be on building simple games such as Pong, Blackjack and Asteroids.

These files were designed to help students with very little or no computing background learn the basics of building simple interactive applications. My language of choice, Python, is an easy-to learn, high-level computer language that is used in many of the computational courses offered on Coursera. To make learning Python easy, I use a new browser-based programming environment (codeskultor) that makes developing interactive applications in Python really simple. These applications will involve windows whose contents are graphical and respond to buttons, the keyboard and the mouse.
